# BulkingTracker
Um app de rotina focado exclusivamente em ganho de massa e hipertrofia. Em vez de focar em emagrecimento, o app ajuda o usuário a registrar o superávit calórico, bater as metas de suplementação diária e acompanhar a evolução na balança ao longo dos meses para atingir a meta de peso até o fim do ano.
